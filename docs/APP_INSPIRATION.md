# App Inspiration

This page maps the apps named in the project request to feature ideas. AetherTune does not copy proprietary code, branding, private APIs, or copyrighted assets from any app.

| App | Feature ideas to consider |
|---|---|
| Kreate | Modern YouTube Music-style UX, cache/download, Android integrations, pitch/tempo concepts. |
| OpenTune | Clean Material 3 experience, simple YT Music-style navigation, offline-first expectations. |
| InnerTune | Search, lyrics, background playback, Android Auto-style integration goals. |
| SimpMusic | Music/video/podcast content grouping and rich discovery pages. |
| ArchiveTune / Archievetune | Archive/open-catalog provider concept. |
| Spotube | Metadata/playback separation and cross-platform account-light design. |
| Echo Music | Extension/provider architecture. |
| Namida | Strong local-library experience and polished player screen. |
| PipePipe | Privacy-first media browsing and background audio. |
| NewPipe | No forced account, subscriptions, lightweight playback UX. |
| LibreTube | Privacy-respecting frontend model. |
| Musify | Lightweight music streaming/player UX. |
| AuraMusic | TV/cast-friendly future ideas. |
| Bloomee Tunes | Multiple provider plugins. |
| Gyawun Music | Simple music-first browsing. |
| Music You | Minimal music UI. |
| Muzza | Material mobile player ideas. |
| SoundPod | Lightweight source/player model. |
| NouTube | Combined YouTube and YouTube Music-style browsing concept. |
| Grayjay | Multi-platform following/subscriptions concept. |
| OuterTune | Local + online library combination. |
| ViTune | ViMusic-style simple player model. |
| RiMusic | Rich music-client options and Android focus. |
| Harmony Music | Cross-platform FOSS music app direction. |
| YMusic | YouTube-audio-style controls; no closed-source behavior copied. |
| YouTube Music official | Polished discovery/library/queue UX; proprietary features not cloned. |
| Flow | Simple, fast music-player flow. |
| MetroList | Music workspace landing-page style, background/cache/sleep-timer feature expectations. |
